# PA-SC-002 Minimum Experiment v0.1

本实验只服务于一个问题：把 `PA-SC-002` 从“策略卡”推进到“能不能跑第一轮最小回测”。

## 实验目标

- 证明 `PA-SC-002` 能否被明确成一版可测试规则。
- 在最小范围里看它在成本后是否有基本正期望的可能。
- 识别它更像在哪些时段/环境赚钱，哪些地方最容易亏。
- 给出单一结论：保留、修改后重测，或淘汰。

## 为什么先测 `SPY 5m`

- `SPY` 是流动性最稳定、点差和滑点相对友好的指数 ETF，适合先做最小 probe。
- `5m` 是 `PA-SC-002` 与 `PA-SC-009` 当前最优先的日内测试周期之一。
- 免费 public intraday 数据在近 60 天窗口内可拿到 `SPY 5m` regular-session 样本，复杂度最低。
- 如果 `SPY 5m` 在成本后都完全站不住，再直接扩到高波动个股意义不大。

| 项目 | 设定 |
|---|---|
| 标的 | `SPY` |
| 周期 | `5m` |
| 时段 | `regular session only` |

| 完整 session 数 | `40` |

## 策略与过滤器

| 项目 | 设定 |
|---|---|
| 主策略 | `PA-SC-002` breakout + follow-through continuation |
| 过滤器 | `PA-SC-009 filter v0.1` |
| 过滤器角色 | 只做 `negative veto`，不做正向评分 |
| Follow-through 窗口 | breakout 后 `1-2` 根 `5m` bar |
| 默认止盈 | 固定 `1R` |
| 默认止损 | breakout bar / 被突破边界外侧结构止损 |

## 成本和滑点假设

| 项目 | 假设 |
|---|---|
| 假设本金 | `25,000 USD` |
| 单笔风险预算 | `100 USD` |
| 仓位计算 | 先按 `100 USD / risk_per_share` 算可买股数，再受账户剩余可买股数限制 |
| 佣金 | `0` |
| 滑点 | `1bp + 1 tick/side` |
| 执行方式 | entry / exit 都按最不利方向计入 |
| 风险说明 | 小 stop breakout 会被摩擦明显侵蚀，所以必须看净 R，而不是裸价格路径 |

## 样本切分

| 切分 | 区间 | 说明 |
|---|---|---|
| 样本内 | `2026-02-20 -> 2026-03-19` | 先看规则能否跑通 |
| 验证集 | `2026-03-20 -> 2026-04-02` | 看结果是否至少不崩 |
| 样本外 | `2026-04-06 -> 2026-04-17` | 看离开样本内后是否还能保住边际 |

## 最低交易数要求

需要分两层看：

| 层级 | 要求 | 用途 |
|---|---|---|
| 最小 probe 门槛 | 总交易数 `>= 60`，验证集和样本外各 `>= 15` | 判断这轮结果是否至少有初步参考价值 |
| 正式 promotion 门槛 | 总交易数 `>= 100`，验证集和样本外各 `>= 20` | 只有达到这个门槛，才谈是否进入更严格验证 |

说明：

- 本轮是“最小实验”，不是正式 promotion。
- 如果只达到 probe 门槛，只能下 exploratory 结论，不能宣传为已验证有效。

## 通过标准

| 层级 | 标准 |
|---|---|
| 最小 probe 通过 | 成本后整体 `Expectancy > 0`，样本外 `Profit Factor >= 1.00`，最大回撤不显著失控 |
| 正式 promotion 通过 | 成本后整体 `Expectancy > 0`，整体 `PF >= 1.15`，样本外 `PF >= 1.00`，最大回撤 `<= 12R` |

## 淘汰标准

| 条件 | 含义 |
|---|---|
| 成本后整体和样本外都明显为负，而且过滤器也无改善 | 本版规则没有继续投入价值 |
| 只有极少数孤立案例盈利，无法形成可重复统计 | 不能继续推进 |
| 策略成立完全依赖后验调参 | 不符合本轮最小闭环原则 |

## 什么结果只能算“不足以下结论”

| 情况 | 处理方式 |
|---|---|
| 没达到最小 probe 门槛 | 标记 `insufficient_sample` |
| 缺少事件/新闻排除标签 | 明确写为数据缺口，不伪装成已排除 |
| 只在单一 split 或单一时间段好看 | 只能作为线索，不能算验证通过 |

## 当前已知数据缺口

- 仓库原本没有现成可用的 `SPY 5m` 缓存，需要先补一份 research-only 本地 CSV。
- 当前仓库没有可直接复用的 `SPY 5m` 新闻/财报标签，因此本轮不能排除宏观新闻窗口。
- 这意味着本轮结果能回答“规则能不能客观化、第一轮结果如何”，但不能回答“是否已足够稳健可推广”。

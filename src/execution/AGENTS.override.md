# AGENTS.override.md

本目录是高风险模块，规则优先于仓库根目录通用规则。

## 默认安全前提
- 默认禁止实盘。
- 未经用户明确批准，不得开启真实自动下单。
- 浏览器方案仅可用于验证或模拟，不得默认承担真实执行链路。

## 改动边界
- 不得绕过风控直接下单。
- 不得省略日志、审计字段、异常处理、幂等性和重复下单保护。
- 不得把模拟盘配置与实盘配置混用。
- 不得擅自修改默认仓位、默认杠杆、默认订单类型和默认重试策略。

## 每次改动必须验证
- 正常下单路径
- 失败路径
- 网络或接口异常路径
- 风控拦截路径
- 模拟盘路径
- 市场关闭路径
- 重复信号路径

## 必须人工复核的改动
- 新增真实下单路径
- 修改实盘开关
- 修改订单类型映射
- 修改执行重试机制
- 修改与券商认证、签名、权限相关逻辑

## 完成定义附加项
- 模拟盘或等效验证通过
- 审计日志字段完整
- 未默认开启实盘
- 变更说明明确写出执行风险影响

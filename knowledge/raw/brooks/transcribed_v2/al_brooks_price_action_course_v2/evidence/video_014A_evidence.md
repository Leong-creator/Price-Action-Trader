## Used In Chapter

- `part1 p871`: `assets/evidence/video_014A/part1_p0871.webp`，趋势中价格主要朝一个方向移动，并被通道容纳；初学者可能看不清通道。
- `part1 p872`: `assets/evidence/video_014A/part1_p0872.webp`，趋势可以小到单根大 K 线，也可以大到覆盖整张图。
- `part1 p873`: `assets/evidence/video_014A/part1_p0873.webp`，主要趋势图表上通常只有一到两个；次要趋势包括波段、腿和回调，可以有很多。
- `part1 p874`: `assets/evidence/video_014A/part1_p0874.webp`，多头趋势由更高高点和更高低点构成，底部是多头趋势线，顶部是趋势通道线。
- `part1 p875`: `assets/evidence/video_014A/part1_p0875.webp`，多头趋势中正确止损常常在主要多头腿底部下方，不能因次要 HL 就过早上调。
- `part1 p876`: `assets/evidence/video_014A/part1_p0876.webp`，多头腿创新高并有大阳线收高后，尤其在失败空头突破之后，可把止损上调到该多头腿下方。
- `part1 p877`: `assets/evidence/video_014A/part1_p0877.webp`，通道衰竭后可能变得更平更宽；跌破前一个更高低点可能只是更宽的多头趋势。
- `part1 p879`: `assets/evidence/video_014A/part1_p0879.webp`，多头趋势跌破前一个更高低点后，可能只是更宽的多头趋势，也可能演变为 TR 或空头；强突破新高后收紧止损。
- `part1 p880`: `assets/evidence/video_014A/part1_p0880.webp`，跌破前一个更高低点并出现强空头腿后，多头止损出局，市场可能转为 TR 或空头趋势。
- `part1 p881`: `assets/evidence/video_014A/part1_p0881.webp`，宽幅多头通道中回调很深，但仍守在最近多头腿底部上方；强空头反转全部失败。
- `part1 p882`: `assets/evidence/video_014A/part1_p0882.webp`，趋势线需要不断重画；通道很少完美，交易者会持续调整通道。
- `part1 p883`: `assets/evidence/video_014A/part1_p0883.webp`，多头趋势中，多头会买入回撤到上一段上涨腿中部的强阴线收盘，并预期反转无法跌破上一个更高低点。
- `part1 p884`: `assets/evidence/video_014A/part1_p0884.webp`，通道底部可从趋势线附近买入向上反转；先用两个顶部画平行线，再向下拖动形成通道。
- `part1 p886`: `assets/evidence/video_014A/part1_p0886.webp`，空头趋势由更低高点和更低低点构成，顶部是空头趋势线，底部是空头趋势通道线。
- `part1 p887`: `assets/evidence/video_014A/part1_p0887.webp`，空头趋势中正确止损在主要空头腿顶部上方；没有新低和强突破前，不要因次要 LH 就下调止损。
- `part1 p888`: `assets/evidence/video_014A/part1_p0888.webp`，空头腿创新低并有大阴线收低后，尤其在失败多头突破之后，可把止损下调到该空头腿顶部上方。
- `part1 p889`: `assets/evidence/video_014A/part1_p0889.webp`，空头通道衰竭后可能变得更平更宽；反弹突破前一个更低高点可能只是更宽的空头趋势。
- `part1 p890`: `assets/evidence/video_014A/part1_p0890.webp`，突破前一个更低高点后，可能只是更宽的空头趋势，也可能已演变为 TR 或多头；强突破新低后收紧止损。
- `part1 p891`: `assets/evidence/video_014A/part1_p0891.webp`，强多头腿突破后空头止损出局，市场可能转为 TR 或多头趋势。
- `part1 p892`: `assets/evidence/video_014A/part1_p0892.webp`，空头趋势中强势多头反转仍可能失败，只要回调保持在最近空头腿顶部下方，就预期形成更低高点。
- `part1 p893`: `assets/evidence/video_014A/part1_p0893.webp`，空头在所有强势反弹中卖出，预期反转尝试无法突破最近更低高点顶部，并在空头趋势线附近寻找反转下跌。
